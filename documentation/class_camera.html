Detailed Description

WARNING: Position of the View is located on the TOP-LEFT of the view This class represent a view for the scene.

It is considered as a concrete view of the scene used to display elements onto the window.

Camera::Camera ( )

Default constructor.

A Scene can adapt one or many cameras to display the elements. A camera is defined by a 'resolution' and a view port which specify the area inside the window where the camera should display its rendereding.
