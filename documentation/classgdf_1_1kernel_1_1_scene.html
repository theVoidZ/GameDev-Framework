Detailed Description

The Scene class is the base class for all scenes.

The scene class provides a way for creating and organizing several items in a tree strucutre.

This class serves as a container for GameObject, allowing theirs initialization, updating and rendering.

Note
Scene inherits from HierarchicalContainer which provides it with a Component Container.

virtual void gdf::kernel::Scene::init ( )
finalprotectedvirtual

Initializes the scene.

This method does:

  • Creates and initializes the root node (GameObject)
  • Invokes on_init int order Initialize an overridden scene.
  • Initializes the components of the Scene
  • Loads all the resources needed by the scene's objects
  • Constructs and initializes the scene's objects.
Note
This method cannot be overridden.
