void ResourceManager::unload_unused_resources ( )

Unloads all unused resources.

Every resource of the resource manager that is not actually used is removed.

Only shared resource are effected by this method, because non-shared resource are not tracked after they clone themselves.

A resource is unused when the std::shared_ptr<T>::use_count is equal to at most one. Means, there is one unqiue reference pointing to the object, which is the one helo by the resource manager.
